# Job Application Assistant

Automated job scraping, resume optimization, and application tracking system.

## Modules

1. **Job Scraper** - Scrape jobs from LinkedIn, company sites
2. **Resume Parser** - Parse LaTeX resumes
3. **Bullet Manager** - Manage resume bullets
4. **ATS Scorer** - Score resume against job descriptions
5. **Job Fit Scorer** - Evaluate candidate-job fit

## Setup

pip install -r requirements.txt
python -c "import nltk; nltk.download('punkt'); nltk.download('punkt_tab'); nltk.download('stopwords'); nltk.download('wordnet')"
